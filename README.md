## Run dev

```bash
  yarn docker:dev # Docker
  # or
  yarn dev
```

## Run lint

```bash
  yarn lint
```

## Start storybook

```bash
  yarn storybook
```

## Dependencies

- chakra-ui
- axios
- nookies
- yup
- storybook
- react-input-mask
- react-hook-form
- react-query
